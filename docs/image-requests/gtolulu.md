# 📷 GTOlulu 소개 페이지 이미지 요청

안녕하세요. lulu.ai 브랜드 페이지(natris-web) 내 **GTOlulu(GTO루루) 상세 페이지** 제작을 위해 다음 이미지 자산을 요청드립니다.

첨부된 스크린샷의 빨간 박스 표시 영역에 제공 이미지가 들어갑니다.

- **서비스 ID:** `gtolulu`
- **브랜드 컬러:** `#8b5cf6`
- **축:** TECH
- **태그:** AI기반 GTO솔버

---

## 1. 메인 히어로 이미지 (1장) — 필수

- **용도:** 상세 페이지 상단 키비주얼 + 메인 페이지 서비스 카드 썸네일
- **사이즈:** 1600×1000px 이상 (4:3 비율)
- **형식:** PNG 또는 JPG
- **제안 내용:**
  - GTO 분석 화면 / 레인지 차트 UI 목업
  - AI 코치 / 데이터 시각화 그래픽
  - 포커 + AI의 결합을 시각화한 키비주얼
- **파일명:** `gtolulu-hero.png`
- **참고 첨부:** `01-hero-location.png`

## 2. 갤러리 이미지 (3장) — 필수

- **용도:** 상세 페이지 하단 Gallery 섹션 (4:5 세로형 3카드)
- **사이즈:** 800×1000px 이상 (4:5 비율)
- **형식:** PNG 또는 JPG
- **제안 구성:**
  1. GTO 분석 / 최적 액션 제안 화면
  2. 핸드 리뷰 / 상황별 복기 UI
  3. 레인지 시뮬레이션 / 전략 리포트 화면
- **파일명:** `gtolulu-gallery-1.png` ~ `gtolulu-gallery-3.png`
- **참고 첨부:** `02-gallery-location.png`

## 3. 기능별 스크린샷 (4장) — 선택

- **용도:** Features 섹션 각 행 오른쪽 시각 자료
- **사이즈:** 1200×900px 이상 (4:3 비율)
- **현재 상태:** 임시 SVG 일러스트로 대체 중
- **기능 항목:**
  1. GTO 분석 — 게임이론 최적 전략 기반 분석 UI
  2. 핸드 리뷰 — 플레이한 핸드 복기 화면
  3. 레인지 시뮬레이션 — 핸드 레인지 확률 차트
  4. AI 코칭 — 개인 패턴 학습·맞춤 훈련 제안 UI
- **파일명:** `gtolulu-feature-1.png` ~ `gtolulu-feature-4.png`
- **참고 첨부:** `03-features-location.png`

## 4. 로고 — 선택

- **형식:** SVG 또는 고해상 PNG (투명 배경)
- **사이즈:** 512×512px 이상
- **버전:** 메인 컬러 + 단색(흰/검) 각 1종
- **파일명:** `gtolulu-logo.svg`
