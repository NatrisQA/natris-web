## 1. 메인 히어로 이미지 (1장) — 필수

- **용도:** 상세 페이지 상단 키비주얼 + 메인 페이지 서비스 카드 썸네일
- **사이즈:** 1600×1000px 이상 (4:3 비율)
- **형식:** PNG 또는 JPG
- **제안 내용:**
  - 클럽 메인 페이지·멤버 리스트 UI 목업
  - 소모임(홀덤펍·동호회·스터디) 다양성 시각화
  - 사람·네트워크 아이콘을 결합한 키비주얼
- **파일명:** `moitto-hero.png`
- **참고 첨부:** `01-hero-location.png`

## 2. 갤러리 이미지 (3장) — 필수

- **용도:** 상세 페이지 하단 Gallery 섹션 (4:5 세로형 3카드)
- **사이즈:** 800×1000px 이상 (4:5 비율)
- **형식:** PNG 또는 JPG
- **제안 구성:**
  1. 클럽 홈 / 소모임 상세 페이지
  2. 멤버 관리 / 역할 설정 화면
  3. 정산 리포트 / 포인트·랭킹 화면
- **파일명:** `moitto-gallery-1.png` ~ `moitto-gallery-3.png`
- **참고 첨부:** `02-gallery-location.png`

## 3. 기능별 스크린샷 (4장) — 선택

- **용도:** Features 섹션 각 행 오른쪽 시각 자료
- **사이즈:** 1200×900px 이상 (4:3 비율)
- **현재 상태:** 임시 SVG 일러스트로 대체 중
- **기능 항목:**
  1. 소모임 운영 — 클럽 개설·관리 대시보드
  2. 멤버 관리 — 가입 승인/역할/활동 이력 UI
  3. 포인트·랭킹 — 활동 기반 포인트 시스템 화면
  4. 자동 정산 — 회비 수납 / 정산 리포트 화면
- **파일명:** `moitto-feature-1.png` ~ `moitto-feature-4.png`
- **참고 첨부:** `03-features-location.png`

## 4. 로고 — 선택

- **형식:** SVG 또는 고해상 PNG (투명 배경)
- **사이즈:** 512×512px 이상
- **버전:** 메인 컬러 + 단색(흰/검) 각 1종
- **파일명:** `moitto-logo.svg`
