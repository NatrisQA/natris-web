# 📷 PokerLulu 소개 페이지 이미지 요청

안녕하세요. lulu.ai 브랜드 페이지(natris-web) 내 **PokerLulu 상세 페이지** 제작을 위해 다음 이미지 자산을 요청드립니다.

첨부된 스크린샷의 빨간 박스 표시 영역에 제공 이미지가 들어갑니다.

- **서비스 ID:** `pokerlulu`
- **브랜드 컬러:** `#fbbf24`
- **축:** GAME
- **태그:** 소셜 포커 플랫폼

---

## 1. 메인 히어로 이미지 (1장) — 필수

- **용도:** 상세 페이지 상단 키비주얼 + 메인 페이지 서비스 카드 썸네일
- **사이즈:** 1600×1000px 이상 (4:3 비율)
- **형식:** PNG 또는 JPG
- **제안 내용:**
  - 소셜 포커 테이블 화면 + 디바이스 목업
  - 포커 칩·카드 그래픽 + 토너먼트 브랜딩
  - 정식 등급분류(GRAC) 뱃지를 부각시킬 수 있는 구성
- **파일명:** `pokerlulu-hero.png`
- **참고 첨부:** `01-hero-location.png`

## 2. 갤러리 이미지 (3장) — 필수

- **용도:** 상세 페이지 하단 Gallery 섹션 (4:5 세로형 3카드)
- **사이즈:** 800×1000px 이상 (4:5 비율)
- **형식:** PNG 또는 JPG
- **제안 구성:**
  1. 로비 / 테이블 선택 화면
  2. 핸드 플레이 진행 화면
  3. 토너먼트 브래킷 / 랭킹 화면
- **파일명:** `pokerlulu-gallery-1.png` ~ `pokerlulu-gallery-3.png`
- **참고 첨부:** `02-gallery-location.png`

## 3. 기능별 스크린샷 (4장) — 선택

- **용도:** Features 섹션 각 행 오른쪽 시각 자료
- **사이즈:** 1200×900px 이상 (4:3 비율)
- **현재 상태:** 임시 SVG 일러스트로 대체 중
- **기능 항목:**
  1. 소셜 포커 — 프라이빗 룸 / 친구 매칭 화면
  2. 토너먼트 시스템 — 일일 토너먼트 / 시즌 챔피언십 UI
  3. 실시간 중계 — 관전 모드 / 라이브 중계 화면
  4. 정식 등급분류 — GRAC 인증 관련 배너 또는 소개 컷
- **파일명:** `pokerlulu-feature-1.png` ~ `pokerlulu-feature-4.png`
- **참고 첨부:** `03-features-location.png`

## 4. 로고 — 선택

- **형식:** SVG 또는 고해상 PNG (투명 배경)
- **사이즈:** 512×512px 이상
- **버전:** 메인 컬러 + 단색(흰/검) 각 1종
- **파일명:** `pokerlulu-logo.svg`
