# 📷 TubeLuLu 소개 페이지 이미지 요청

안녕하세요. lulu.ai 브랜드 페이지(natris-web) 내 **TubeLuLu(투베루루) 상세 페이지** 제작을 위해 다음 이미지 자산을 요청드립니다.

첨부된 스크린샷의 빨간 박스 표시 영역에 제공 이미지가 들어갑니다.

- **서비스 ID:** `tubelulu`
- **브랜드 컬러:** `#ec4899`
- **축:** TECH
- **태그:** 지식기반 팬덤교류 플랫폼

---

## 1. 메인 히어로 이미지 (1장) — 필수

- **용도:** 상세 페이지 상단 키비주얼 + 메인 페이지 서비스 카드 썸네일
- **사이즈:** 1600×1000px 이상 (4:3 비율)
- **형식:** PNG 또는 JPG
- **제안 내용:**
  - 전문가 프로필·구독 UI 목업
  - 지식 콘텐츠 피드·Q&A 요소
  - 팬덤·커뮤니티 연결 그래픽
- **파일명:** `tubelulu-hero.png`
- **참고 첨부:** `01-hero-location.png`

## 2. 갤러리 이미지 (3장) — 필수

- **용도:** 상세 페이지 하단 Gallery 섹션 (4:5 세로형 3카드)
- **사이즈:** 800×1000px 이상 (4:5 비율)
- **형식:** PNG 또는 JPG
- **제안 구성:**
  1. 전문가 프로필 / 구독 페이지
  2. 인사이트 콘텐츠 피드 화면
  3. 유료 Q&A / 팬덤 커뮤니티 화면
- **파일명:** `tubelulu-gallery-1.png` ~ `tubelulu-gallery-3.png`
- **참고 첨부:** `02-gallery-location.png`

## 3. 기능별 스크린샷 (4장) — 선택

- **용도:** Features 섹션 각 행 오른쪽 시각 자료
- **사이즈:** 1200×900px 이상 (4:3 비율)
- **현재 상태:** 임시 SVG 일러스트로 대체 중
- **기능 항목:**
  1. 전문가 구독 — 프로/전략가/코치 구독 UI
  2. 인사이트 큐레이션 — 심층 분석/전략 콘텐츠 피드
  3. 유료 Q&A — 질문 등록·프리미엄 답변 화면
  4. 팬덤 커뮤니티 — 토론/정보 교환 공간
- **파일명:** `tubelulu-feature-1.png` ~ `tubelulu-feature-4.png`
- **참고 첨부:** `03-features-location.png`

## 4. 로고 — 선택

- **형식:** SVG 또는 고해상 PNG (투명 배경)
- **사이즈:** 512×512px 이상
- **버전:** 메인 컬러 + 단색(흰/검) 각 1종
- **파일명:** `tubelulu-logo.svg`
