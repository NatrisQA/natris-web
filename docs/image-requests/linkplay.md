# 📷 LinkPlay 소개 페이지 이미지 요청

안녕하세요. lulu.ai 브랜드 페이지(natris-web) 내 **LinkPlay 상세 페이지** 제작을 위해 다음 이미지 자산을 요청드립니다.

첨부된 스크린샷의 빨간 박스 표시 영역에 제공 이미지가 들어갑니다.

- **서비스 ID:** `linkplay`
- **브랜드 컬러:** `#06b6d4`
- **축:** GAME
- **태그:** 방송·게임 인터랙티브 플랫폼

---

## 1. 메인 히어로 이미지 (1장) — 필수

- **용도:** 상세 페이지 상단 키비주얼 + 메인 페이지 서비스 카드 썸네일
- **사이즈:** 1600×1000px 이상 (4:3 비율)
- **형식:** PNG 또는 JPG
- **제안 내용:**
  - 라이브 방송 화면 + 시청자 참여 UI 오버레이
  - 스트리머·시청자 연결을 시각화한 이미지
  - LIVE 뱃지, 채팅/리액션 요소 포함 가능
- **파일명:** `linkplay-hero.png`
- **참고 첨부:** `01-hero-location.png`

## 2. 갤러리 이미지 (3장) — 필수

- **용도:** 상세 페이지 하단 Gallery 섹션 (4:5 세로형 3카드)
- **사이즈:** 800×1000px 이상 (4:5 비율)
- **형식:** PNG 또는 JPG
- **제안 구성:**
  1. 라이브 방송 중 미니게임 플레이 화면
  2. 스트리머 대시보드 / 게임 설정 화면
  3. 시청자 참여·결과 화면
- **파일명:** `linkplay-gallery-1.png` ~ `linkplay-gallery-3.png`
- **참고 첨부:** `02-gallery-location.png`

## 3. 기능별 스크린샷 (4장) — 선택

- **용도:** Features 섹션 각 행 오른쪽 시각 자료
- **사이즈:** 1200×900px 이상 (4:3 비율)
- **현재 상태:** 임시 SVG 일러스트로 대체 중
- **기능 항목:**
  1. 실시간 매칭 — 방송 중 시청자 매칭 UI
  2. 크리에이터 연동 — 스트리머 대시보드 (게임 설정/참가자 관리)
  3. 시청자 참여형 게임 — 채팅 연동 참가 UI
  4. 다양한 미니게임 — 게임 라이브러리 / 썸네일 모음
- **파일명:** `linkplay-feature-1.png` ~ `linkplay-feature-4.png`
- **참고 첨부:** `03-features-location.png`

## 4. 로고 — 선택

- **형식:** SVG 또는 고해상 PNG (투명 배경)
- **사이즈:** 512×512px 이상
- **버전:** 메인 컬러 + 단색(흰/검) 각 1종
- **파일명:** `linkplay-logo.svg`
