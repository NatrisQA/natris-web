# 📷 ShuffleUp 소개 페이지 이미지 요청

안녕하세요. lulu.ai 브랜드 페이지(natris-web) 내 **ShuffleUp 상세 페이지** 제작을 위해 다음 이미지 자산을 요청드립니다.

첨부된 스크린샷의 빨간 박스 표시 영역에 제공 이미지가 들어갑니다.

- **서비스 ID:** `shuffleup`
- **브랜드 컬러:** `#3b82f6`
- **축:** COMMUNITY
- **태그:** 대회 운영 솔루션 플랫폼

---

## 1. 메인 히어로 이미지 (1장) — 필수

- **용도:** 상세 페이지 상단 키비주얼 + 메인 페이지 서비스 카드 썸네일
- **사이즈:** 1600×1000px 이상 (4:3 비율)
- **형식:** PNG 또는 JPG
- **제안 내용:**
  - 오프라인 포커 대회 운영 대시보드 목업
  - 블라인드 타이머 / 테이블 배치 UI
  - 실시간 대회 진행 상황 그래픽
- **파일명:** `shuffleup-hero.png`
- **참고 첨부:** `01-hero-location.png`

## 2. 갤러리 이미지 (3장) — 필수

- **용도:** 상세 페이지 하단 Gallery 섹션 (4:5 세로형 3카드)
- **사이즈:** 800×1000px 이상 (4:5 비율)
- **형식:** PNG 또는 JPG
- **제안 구성:**
  1. 대회 운영 메인 대시보드
  2. 블라인드 타이머 / 테이블 배정 화면
  3. 정산 리포트 / 상금 분배 화면
- **파일명:** `shuffleup-gallery-1.png` ~ `shuffleup-gallery-3.png`
- **참고 첨부:** `02-gallery-location.png`

## 3. 기능별 스크린샷 (4장) — 선택

- **용도:** Features 섹션 각 행 오른쪽 시각 자료
- **사이즈:** 1200×900px 이상 (4:3 비율)
- **현재 상태:** 임시 SVG 일러스트로 대체 중
- **기능 항목:**
  1. 토너먼트 관리 — 참가 등록/테이블 배정/진행 현황
  2. 블라인드 타이머 — 레벨별 블라인드 구조 + 타이머 UI
  3. 시딩 시스템 — 랜덤/시드 기반 매칭 브래킷
  4. 자동 정산 — 상금 분배·참가비 정산·수익 리포트
- **파일명:** `shuffleup-feature-1.png` ~ `shuffleup-feature-4.png`
- **참고 첨부:** `03-features-location.png`

## 4. 로고 — 선택

- **형식:** SVG 또는 고해상 PNG (투명 배경)
- **사이즈:** 512×512px 이상
- **버전:** 메인 컬러 + 단색(흰/검) 각 1종
- **파일명:** `shuffleup-logo.svg`
